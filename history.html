Java Basics

History of Java

Java was developed by James Gosling at Sun Microsystems in 1995. It was created to be a platform-independent, object-oriented programming language that can run on any device using the Java Virtual Machine (JVM).

Java quickly became popular due to its portability, security, and robustness. Today, it is used in web applications, mobile apps, enterprise software, and more.

Class

A class is a blueprint or template for creating objects. It defines properties (variables) and behaviors (methods) that the objects created from the class can have.

Example: A class named Car might have properties like color and model, and behaviors like drive() and stop().

Object

An object is an instance of a class. It contains actual values for the properties defined in the class and can use the behaviors (methods) provided by the class.

Example: If Car is the class, then myCar could be an object with color “red” and model “Toyota”.

Variables

Variables are used to store data in a program. Each variable has a type which defines what kind of data it can hold, such as integers, decimal numbers, text, or boolean values (true/false). Variables allow programs to store, change, and use data during execution.
